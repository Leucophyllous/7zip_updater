# 7zip_updater

7-Zip を自動でアップデートする Windows 用ツールです。

## 概要

- 7-Zip のバージョンチェック・ダウンロード・インストールを自動化します。
- Python + Tkinter (PyInstaller) でビルドされた単体の実行ファイル (`7zip_updater.exe`) として配布しています。

## 使い方

1. [Releases](../../releases) または本リポジトリから `7zip_updater.exe` をダウンロードします。
2. ダブルクリックで実行します（Windows のみ対応）。
3. 画面の指示に従ってアップデートを実行してください。

## 注意事項

- 本リポジトリにはソースコードが含まれていません（ビルド済みバイナリのみ）。
- 実行には Windows 環境が必要です。
- 未知の実行ファイルを実行する際は、事前にウイルス対策ソフトやオンラインスキャン（VirusTotal 等）での確認を推奨します。
